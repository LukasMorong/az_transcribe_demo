# AMAZON Transcribe DEMO

## INSTALL

```
pip install -r requirements
```

## RUN

```
python app.py
```

https://pypi.org/project/amazon-transcribe

## SETUP AWS

1. create AWS account (credit card required)
2. install aws-cli
3. configure credentials

```
aws configure
```

Create acess key
